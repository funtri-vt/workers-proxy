# workers-websocket-proxy
A secure, edge-computed web proxy built entirely on Cloudflare Workers. It uses wildcard DNS routing to enforce strict Same-Origin Policy (SOP) isolation, and SQLite storage (Cloudflare D1) to securely manage session state and cookies. Protected by Cloudflare Zero Trust, it allows authenticated users to seamlessly browse the web and establish WebSocket connections without compromising native browser sandboxing or exposing credentials to client-side scripts.
